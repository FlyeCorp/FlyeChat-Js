# FlyeChat JS

FlyeChat JS is a version a FlyeChat built of the Vue Js framework.

## Project setup
```
npm install
```

### Run API
```
npm run api
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
